进程和线程的区别:

  • 进程是最小的内存分配单位
  • 线程是操作系统调度的最小单位
  • 进程内至少包含一个线程
  • 进程中可以开启多个线程
  • 开启一个线程所需要的时间远远小于开启一个进程

控制台什么都没有输出，这是因为主线程立即运行完毕，线程t被设置成守护线程，也随着主线程的结束而结束。
主线程在其他__非守护进程__运行完毕之后才算执行完毕。因为主线程的结束意味着进程的结束，进程整体的资源都将被回收，而进程必须必须保证非守护线程都运行完毕后才能结束。
